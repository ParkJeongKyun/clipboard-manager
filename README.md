# Clipboard Manager

Clipboard Manager is a macOS application designed to enhance clipboard functionality by providing a user-friendly interface for managing clipboard history. This application allows users to view, copy, and manage their clipboard items efficiently.

## Features

- **Clipboard History**: View a history of all items copied to the clipboard.
- **Easy Access**: Quickly access clipboard items from the main interface.
- **User-Friendly Interface**: Intuitive design for seamless user experience.
- **Clipboard Management**: Copy, delete, and manage clipboard items with ease.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/clipboard-manager.git
   ```
2. Open the project in Xcode:
   ```
   open clipboard-manager/ClipboardManager.xcodeproj
   ```
3. Build and run the application.

## Usage

- Launch the application to view the clipboard history.
- Click on any item to copy it back to the clipboard.
- Use the interface to manage your clipboard items.
